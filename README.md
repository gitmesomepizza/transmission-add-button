# transmission-add-button
Add a custom toolbar button to transmission web interface.
<hr>
Running the patch-transmission.sh file should patch transmission to add a custom button to the toolbar.<br>
But review code in patch-transmission.sh before running to ensure everything looks good.<br>
Tested on Ubuntu Server 16.04.4 LTS<br>
